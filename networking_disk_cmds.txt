df (Disk Free)
Displays disk space usage for file systems.
Common options:
df -h: Shows human-readable output (e.g., sizes in GB, MB).
-------------------------------------------------------------------------------------------
du (Disk Usage)
Displays the disk usage of files and directories.
Common options:
du -h: Human-readable output.
du -sh: Shows a summary for a directory.
